# Planejamento Básico de Viagem Questão do lab

Programa em Python para estimar custos e organizar informações de uma viagem.

## Arquivos
- `funcoes_viagem.py` — funções de cálculo
- `principal_viagem.py` — lógica principal do programa

## Funcionalidades
- Calcula o valor da passagem com taxa de bagagem opcional
- Calcula a hospedagem por diária com taxa extra opcional
- Converte duração da viagem de horas para dias e horas restantes
- Gera orçamento com custo fixo, custo extra e total geral
