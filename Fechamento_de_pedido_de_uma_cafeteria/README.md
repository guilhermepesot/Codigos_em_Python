# Fechamento de Pedido de uma Cafeteria - Questão do lab

Sistema simples em Python para calcular o fechamento de pedidos de uma cafeteria.

## Arquivos
- `funcoes_cafeteria.py` — funções de cálculo
- `principal_cafeteria.py` — lógica principal do programa

## Funcionalidades
- Calcula o preço do café com acréscimo por tamanho (P/M/G)
- Aplica desconto no acompanhamento
- Gera resumo do pedido com subtotal, taxa de serviço e total final
