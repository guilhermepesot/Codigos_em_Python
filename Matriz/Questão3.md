# Multiplicação de Matriz por Escalar

Programa em Python que lê uma matriz 3x3 e multiplica todos os elementos por um número escolhido pelo usuário.

## Arquivos
- `escalar.py` — código principal

## Funcionalidades
- Lê os valores de cada posição da matriz via input
- Recebe um número multiplicador do usuário
- Exibe a matriz com todos os elementos multiplicados pelo escalar
