# Sistema de Relatório de Inscrições em Evento

Aplicação em Python para consolidar inscrições em um evento temático com oficinas e atividades extras.

## Arquivos
- `funcoes_evento.py` — funções auxiliares e de processamento
- `principal_evento.py` — lógica principal do programa

## Funcionalidades
- Calcula o valor base do ingresso por tipo (regular, vip, estudante)
- Adiciona extras como oficinas e material complementar
- Aplica cupom de desconto e taxa administrativa
- Classifica o perfil de inscrição de cada participante
- Gera relatório consolidado para 3 participantes
