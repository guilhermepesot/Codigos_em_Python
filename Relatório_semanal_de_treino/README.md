# Relatório Semanal de Treino

Programa em Python para registrar e analisar o desempenho semanal de um aluno de academia.

## Arquivos
- `funcoes_treino.py` — funções de cálculo
- `principal_treino.py` — lógica principal do programa

## Funcionalidades
- Calcula calorias gastas com bônus opcional
- Calcula tempo total do treino com aquecimento
- Converte tempo total em horas e minutos
- Verifica se a meta semanal de calorias foi atingida
- Gera relatório final com resumo do desempenho
